

## QA Study Buddy - Clean & Modern Chat UI

### Layout
- **Header**: App name "QA Study Buddy" with a clean logo/icon, subtle bottom border, and a teal/emerald accent color
- **Left Sidebar** (~280px): Clean white card with subtle shadow, collapsible
- **Main Chat Area**: Full remaining width with a light gray background

### Sidebar Features
1. **Upload Documents Section**
   - Styled drag-and-drop zone with dashed border and file icon animation on hover
   - "Select PDF files" button with emerald/teal gradient
   - File count badge and "Process PDFs" button
   
2. **YouTube Video URLs Section**
   - Clean input field with rounded corners + "Add" button with pop color
   - List of added URLs with remove option
   - "Process Videos" button with count badge

### Chat Area
- **Empty state**: Centered illustration/icon with friendly message and subtle animation
- **Messages**: Chat bubbles with distinct user (right-aligned, colored) and bot (left-aligned, white card with shadow) styling
- **Typing indicator**: Animated dots when "processing"

### Bottom Input Bar
- Fixed to bottom of chat area with slight elevation/shadow
- Mark type dropdown selector (styled select with rounded corners)
- Wide text input with placeholder "Ask a question..."
- Teal/emerald "Send" button with hover animation

### Design Details
- **Color scheme**: White/light gray base, teal/emerald (#0d9488) as primary accent, soft shadows
- **Typography**: Clean sans-serif, proper hierarchy
- **Animations**: Subtle hover effects on buttons, smooth transitions on sidebar toggle
- **Border radius**: Generous rounded corners (12-16px) for cards and inputs
- **Spacing**: Generous padding and whitespace for a breathable layout
